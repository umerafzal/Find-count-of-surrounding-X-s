# Find-count-of-surrounding-X-s

Given a 2D array, solve the question: Find the count of surrounding ’X’s on every Index of an array except where index contains an ‘X’.
Input:
[ [ X, 0 , 0 ], [ 0, 0, 0 ], [ X, X, 0 ] ]
Output: 
[ [X, 1 , 0 ], [ 3, 3, 1 ], [ X, X, 1 ] ]
